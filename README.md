# Portal da Ciência Cidadã

## Sobre

Este repositório contém o front-end do Portal da Ciência Cidadã, desenvolvido para envolver a comunidade com iniciativas científicas e facilitar a participação em atividades de ciência cidadã. O projeto foi criado como parte de uma atividade de extensão.

## Funcionalidades

- Interface responsiva para exibir projetos e observações
- Formulários para submissão de observações/participação
- Visualização e filtragem de dados (listas, tabelas e mapas)
- Integração com APIs para obtenção e envio de dados

## Tecnologias

- React, JavaScript, HTML e CSS (pode ser adaptado conforme a implementação)
- Ferramentas comuns: Node.js, npm/yarn
